### BookStack （书栈）

项目介绍:
> BookStack，基于MinDoc，使用Beego开发的在线文档管理系统，功能类似Gitbook和看云。

开源协议：Apache 2.0

最新版本：v1.4

技术栈体系：`Beego（Golang）` + `BootStrap` + `MySQL`

支持平台：`Windows`、`Linux`、`Mac`

开源地址

- https://github.com/TruthHun/BookStack
- https://gitee.com/TruthHun/BookStack

## 赞助名单

随着时间的积累，开源项目，可能会越来越多。

一般情况下，每个项目都有赞助方式和赞助二维码。

热心的朋友在对A项目进行赞助的时候，可能在A项目的赞助名单里面看不到自己的赞助公示，因为他对A项目的赞助可能被写在了B项目的赞助名单下了，因为作者在收到赞助的时候，无从得知赞助是来自A项目的还是B项目的.

而且，如果每当有朋友对项目进行了赞助都在项目里更新一次赞助名单的话，`watch(关注)`了开源项目的用户可能就会频繁收到不是项目源码更新而带来的邮件或短信提醒。
